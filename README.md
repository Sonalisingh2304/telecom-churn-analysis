Project Overview

This project analyzes customer churn behavior in a telecom company using Python-based exploratory data analysis.
The goal is to identify key factors causing customers to leave and provide business-driven retention strategies.


Dataset

~7000 telecom customers

Includes demographics, contract type, payment methods, and service usage

Target variable: Churn (Yes/No)

Key Insights

Month-to-month customers have the highest churn rate

Electronic check users are most likely to leave

Customers without tech support/security churn significantly more

Fiber users churn due to expectation gap

Early customer lifecycle is the most critical stage

Business Recommendations

Promote long-term contracts

Encourage automatic payment methods

Improve onboarding experience

Bundle support services

Prioritize high-value customers

Python (Pandas, Matplotlib)
